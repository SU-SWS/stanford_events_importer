Stanford Events Importer 7.x-2.0-beta3, 2012-11-09
--------------------------------------------------
The 7.x-2.0-beta3 release is functionally identical to the 7.x-2.0-beta2 release.
The only changes are to the content of this CHANGELOG.txt file.
Changes since 2.0-beta1 include:
  -improvements to upgrade path from 7.x-1.x
  -changing date field to use the Date Popup functionality
  -additional support for all-day events

Stanford Events Importer 7.x-2.0-beta1, 2012-10-19
--------------------------------------------------
The 7.x-2.0-beta1 release is the first release in the 7.x-2.x branch that should
  be considered stable.
Changes since 2.0-alpha6 include:
  -support for updating from 7.x-1.x branch
  -configuration screen
  -update Drupal database tables with category and organization lists on cron runs
  -bug fixes

Stanford Events Importer 7.x-2.0-alpha6, 2012-10-17
---------------------------------------------------
Changes since 2.0-alpha1 include:
  -Automatically pulling list of categories and organizations from events.stanford.edu
    and presenting those to the user in a dropdown list at node/add/stanford-event-importer
    (no more manual pasting of feed URL)
(-alpha2 through -alpha5 were incremental development releases and likely should not be used by anyone.)
